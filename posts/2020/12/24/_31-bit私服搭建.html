# bit 私服搭建

# 准备

  1. 一台内存 4g 以上的服务器
  2. 安装 docker

# 拉取 docker 镜像

docker pull bitcli/bit
docker pull bitcli/bit-server

# 运行镜像

docker run -it bitcli/bit:latest /bin/bash
docker run -it -p {host-port}:3000 bitcli/bit-server:latest
